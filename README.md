# Raytracer

### Usage
```
Usage: ./raytracer [OPTIONS]
  --cornell-box           load test model
  --load     <file.obj>   load custom model
  --load-box <file.obj>   load custom model, inside the Cornell Box
```

### Controls
```
 Move view:             Move light:             Rotate camera:

     W                      I                         ^
  A  S  D                J  K  L                   <  v  >

  Q --> up               U --> up
  E --> down             P --> down
```
